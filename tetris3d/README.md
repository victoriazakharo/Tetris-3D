# Tetris-3D
Challenging 3d Tetris game using Winapi and OpenGL<br/><br/>
Key settings:<br/>
  Camera movement: WASD<br/>
  Figure rotation: Up<br/>
  Figure movement: Left, Right, Ctrl, Shift<br/>
  Acceleration: Down<br/>
